# primitives

Installed primitives — self-contained dotdir protocols living inside `.gremlin/`. Each bundles its script and its data; read its README before working with one.

- 🔮 glean — `.gremlin/.glean/` — A tiny, file-based protocol for memory and distillation.
- 🦫 groundhog — `.gremlin/.groundhog/` — A tiny, file-based protocol for recurring items.
- 🪡 loom — `.gremlin/.loom/` — A tiny, file-based protocol for planning and tending chains of work.
- 🐉 lore — `.gremlin/.lore/` — A tiny, file-based protocol for durable reference and record.
- 🪺 nestlings — `.gremlin/.nest/` — A tiny, file-based protocol for agents that tend a folder.
